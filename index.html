Welcome-цепочка с интеграцией в AmoCRM

Автоматизация

B2B платформа медицинского туризма — нужно было автоматически прогревать новых партнеров и отслеживать их активность в CRM.

Что сделал

  • Спроектировал и настроил автоматическую welcome-цепочку из 33+ узлов
  • Интегрировал цепочку с AmoCRM — логика следующего письма зависела от действий партнера в системе
  • Настроил персонализацию через переменные с fallback-значениями — письма приходили с именем получателя даже если поле не заполнено
  • Выстроил систему интервалов касаний — 1, 7 и 14 дней в зависимости от поведения
Результат Цепочка работала без ручного участия — новые партнеры автоматически получали релевантные письма в зависимости от своих действий в системе.

Сегментированная коммуникация для трех типов партнеров

Стратегия

Партнерская сеть платформы включала разные типы участников с разными задачами и интересами — единая коммуникация не работала.

Что сделал

  • Выстроил раздельную коммуникацию для трех сегментов — русскоязычные клиники, зарубежные клиники, русскоязычные агентства
  • Разработал отдельный контент под каждый сегмент с учетом роли и задач партнера
  • Англоязычную коммуникацию с зарубежными клиниками вел самостоятельно — без переводчика
  • Внедрил персонализацию обращений с учетом пола и падежа для русскоязычной аудитории
Результат Система покрывала три рынка без привлечения дополнительных специалистов — контент, переводы и коммуникация велись одним человеком.

RFM-анализ партнерской базы

Аналитика

Стандартная методология RFM рассчитана на ecommerce — нужно было адаптировать ее под B2B где нет прямых транзакций.

Что сделал

  • Провел RFM-анализ партнерской базы — адаптировал методологию под B2B специфику
  • Вместо денежных транзакций использовал поведенческие метрики — подписка, регистрация в системе, переходы по ссылкам
  • Сегментировал базу по уровню вовлеченности для персонализации дальнейшей коммуникации
Результат Получил сегментацию базы по реальной активности партнеров — основа для разной коммуникации с активными и неактивными контактами.
